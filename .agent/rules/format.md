# date and time format

The format must be in ISO och close-to-ISO.
* yyyy-mm dd
* hh:MM
* hh:MM:ss
* yyyy-mm-dd hh:MM
* yyyy-mm-dd hh:MM:ss
* ISO 8601

There might be situations where time zone is of importance.  
There must be a setting for toggling between local and UTC time, globally for the whole application.
